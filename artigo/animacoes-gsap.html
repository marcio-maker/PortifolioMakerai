Animações com GSAP e Three.js

28 Fev 2026 15 min leitura Marcio Maker

Tutorial completo para criar animações impressionantes usando as melhores bibliotecas.

GSAP e Three.js são duas das bibliotecas mais poderosas para animações na web. Neste tutorial, vamos criar animações complexas, desde efeitos de scroll até modelos 3D interativos. Código passo a passo e exemplos práticos inclusos.

Tecnologias abordadas

GSAP Three.js Animações WebGL
“Animações não são apenas enfeites, são parte da experiência do usuário.”

Exemplo prático com GSAP

// Timeline GSAP const tl = gsap.timeline({ defaults: { duration: 1, ease: "power3.out" } }); tl.from(".box", { opacity: 0, y: 50, stagger: 0.2 }); tl.from(".title", { opacity: 0, scale: 0.8 }, "-=0.5");
